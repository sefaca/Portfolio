App Mobile

FoodSaver

FoodSaver is a mobile app developed with React Native that helps you manage your fresh food. It uses text recognition and a camera to scan products and monitor best-before dates, helping to reduce food waste.

CookMate

CookMate is a mobile application developed with React Native that allows users to search for recipes according to calorie filters and type of cuisine, integrating Edamam API to obtain personalised recipes.

ExpenseTracker

ExpenseTracker is a React Native app that scans and recognises text from receipts, automatically calculating total expenses. Built with React Native technology and OCR for seamless expense tracking supported with a graphical display.

WeatherAIApp

WeatherAIApp is a weather forecast app developed with React Native and Cohere AI. It offers real-time updates and solutions for your outfit according to the weather in your area.

PadelMatch

It is a project developed for my TFG, which facilitates the organisation of padel matches and the management of bookings. Developed with PHP, MySQL, JavaScript, HTML and CSS, it offers a simple and functional interface.

Currency Tracking Tool

BatataBit is a personal website that offers real-time analysis of cryptocurrencies and currencies. Developed with HTML, CSS, and JavaScript, it integrates APIs for accurate and up-to-date data.

Interactive Restaurant Website

Clone of a restaurant website created to practice my skills. Developed with HTML, CSS, and JavaScript, it features an interactive and responsive design for an optimised user experience.
